# Homebrew Flowbear

Official Homebrew tap for [Flowbear](https://github.com/scttfrdmn/flowbear) - The Anti-Ghostwriter.

## Installation

```bash
brew tap scttfrdmn/flowbear
brew install flowbear
```

## Usage

After installation, run:

```bash
flowbear --help
```

## Updating

```bash
brew update
brew upgrade flowbear
```

## Uninstallation

```bash
brew uninstall flowbear
brew untap scttfrdmn/flowbear
```

## About Flowbear

Flowbear is a privacy-first writing assistant that helps writers organize, refine, and maintain their creative work without generating content for them.

**Define your voice. Refine your voice. Never replace your voice.**
